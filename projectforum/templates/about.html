About Project Forum

Project Forum is the product of a semester long project in the CS169 course at UC Berkeley. Our goal is to produce a contracting job board that both employers and contractors can leverage. This project was put together by Alex, Emily, Sarah, and Cedric.
Special thanks to these various technologies and tools that made this project possible:
  • Web Framework: Django
  • Backend: PostgreSQL
  • Frontend: JQuery, Bootstrap, Sass
  • Notifications System: Mailgun
  • Testing: Selenium
